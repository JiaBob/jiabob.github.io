HTML & CSS Validation

There are some errors occured on HTML among all the HTML pages. The first one is about the web page icon. As it is a famous method to put icon for the website, I will remain this error here. The second error is about javascript loading. Code inside "script defer" will be loadded after DOM tree finished loading. What the validation error said is to put JS code in one more file and use "src" to load it. But for my case, those codes must be inside the same page with HTML. The third error is because my image tag lacks "alt" attribute. This is truely my mistake. But as there are so many image in my gallery, I decided not to fix it at this time. For CSS validation, the error shows that the validator cannot understand CSS variable, which is a valid function of new CSS. I use this function for alternative CSS page. Thus, it is actually not an error.
